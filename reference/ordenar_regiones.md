# Ordenar regiones de Chile geogr\u00e1ficamente

Ordena una columna de regiones de Chile (`nombre_region`) a partir de la
columna `codigo_region`

## Uso

``` r
ordenar_regiones(data)
```

## Argumentos

- data:

  Dataframe que contenga las columnas `nombre_region` y `codigo_region`

## Valor

Dataframe con la variable `nombre_region` en formato factor, ordenado
geogr\u00e1ficamente (de norte a sur)
