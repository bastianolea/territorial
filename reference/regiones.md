# Nombres de regiones de Chile

Vector de texto que contiene el nombre de las 15 regiones de Chile,
ordenadas de norte a sur, tal como aparecen en la tabla
[territorios](https://bastianolea.github.io/territorial/reference/territorios.md).

## Uso

``` r
regiones()
```

## Valor

Vector de nombres de regiones de Chile

## Ejemplos

``` r
regiones()
#>  [1] "Arica y Parinacota"                       
#>  [2] "Tarapacá"

#>  [6] "Valparaíso"

#>  [9] "Maule"                                    
#> [10] "Ñuble"                                    
#> [11] "Biobío"

#> [13] "Los Ríos"                                 
#> [14] "Los Lagos"
```
