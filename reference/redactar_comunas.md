# Redactar una secuencia de comunas en una oración

Para un vector de nombrees de comuna, retorna un texto con las comunas
redactadas y separadas por comas.

## Uso

``` r
redactar_comunas(x)
```

## Argumentos

- x:

  Nombres de comunas

## Valor

Texto con comunas separadas por comas

## Ejemplos

``` r
redactar_comunas(c("Paine", "Isla de Maipo", "Pirque"))
#> Paine, Isla de Maipo y Pirque
```
